Classe
É um modelo para criar objetos
Objeto
É uma instância de uma classe
Método
É uma função definida dentro de uma classe
Atributo
É uma variável definida dentro de uma classe
Construtor
É um método especial usado para inicializar objetos
Herança
Permite que uma classe adquira propriedades de outra classe
Polimorfismo
Permite que objetos sejam tratados como instâncias de sua classe pai
Interface
É um contrato que define métodos que uma classe deve implementar
